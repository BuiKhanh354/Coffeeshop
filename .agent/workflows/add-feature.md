# Thêm Tính Năng Mới

Khi yêu cầu thêm tính năng, hãy cung cấp thông tin sau:

## Template yêu cầu

```
Tính năng: [Tên tính năng]
Mô tả: [Mô tả chi tiết tính năng cần làm]
Vị trí: [Controller/View/Model nào cần thay đổi, nếu biết]
Tham khảo: [Link hoặc hình ảnh tham khảo, nếu có]
```

## Ví dụ yêu cầu tốt

```
Tính năng: Thêm chức năng đánh giá sản phẩm
Mô tả:
- Khách hàng có thể đánh giá sản phẩm từ 1-5 sao
- Hiển thị trung bình đánh giá trên trang sản phẩm
Vị trí: ProductsController, Views/Products/Details.cshtml
Tham khảo: Giống như trang Shopee
```

## Lưu ý

- Yêu cầu càng chi tiết, agent làm càng nhanh và chính xác
- Gộp các yêu cầu liên quan vào một lần hỏi
