# Build Tailwind CSS

// turbo-all

1. Di chuyển đến thư mục web:

```bash
cd d:\githubclone\Coffeeshop\src\CoffeeShop.Web
```

2. Cài đặt dependencies (nếu chưa có):

```bash
npm install
```

3. Build Tailwind CSS (development):

```bash
npm run build:css
```

4. Hoặc watch mode để tự động build khi có thay đổi:

```bash
npm run watch:css
```
