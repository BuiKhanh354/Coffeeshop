# Commit và Push Code

1. Kiểm tra trạng thái git:

```bash
git status
```

2. Thêm tất cả thay đổi:

```bash
git add .
```

3. Commit với message (thay [MESSAGE] bằng mô tả của bạn):

```bash
git commit -m "[MESSAGE]"
```

// turbo 4. Push lên branch dev:

```bash
git push origin dev
```
